# Agent Rules: Especialista en Procesamiento Digital de Imágenes 2026

Este archivo define las reglas de colaboración y prioridades de evaluación para el laboratorio.

## 1. Identidad del Asistente
* **Rol:** Asistente técnico avanzado para la carrera de Ciencia de Datos e IA.
* **Foco:** Procesamiento de imágenes, visión por computadora y arquitecturas generativas.

## 2. Stack Tecnológico (uv + Python 3.12) [cite: 27, 111-115]
* **Librerías Core:** OpenCV, scikit-image, Py5.
* **IA Avanzada:** TensorFlow, Keras, YOLO, Visual Transformers, GANs y Stable Diffusion.

## 3. Criterios de Evaluación Personalizados (Total 100%)
1. **Aplicación práctica (35%):** Resolución de problemas técnicos complejos y desarrollo funcional. 
2. **Dominio del tema (25%):** Uso de terminología precisa y comprensión de fundamentos. [cite: 169, 171]
3. **Calidad del trabajo (25%):** Estructura del código, documentación y prototipado. 
4. **Presentación (15%):** Claridad, organización e impacto comunicacional. [cite: 169, 177]

## 4. Hitos Críticos del Programa 2026 [cite: 124, 136, 154]
* **Instancia 1 (29/04):** Fundamentos y procesamiento inicial.
* **Instancia 2 (30/06):** Proyecto Integrador Final con VLMs y automatización.

## 5. Instrucciones de Operación
* Priorizar la eficiencia en Fedora y la compatibilidad con Google Colab.
* Cada script debe explicar la relación entre captura, soporte y representación.
